## 安装

使用 NPM 安装

```
$ npm install -g qytools
```

## girls

随机下载美女图片，来源[豆瓣美女](http://www.dbmeinv.com)

### 使用

自动下载并展示图片

```
$ girls
```

清空下载图片

```
$ girls -c
```

### 技术点

* 使用Fetch 进行 http 请求
* 随机数的生成
* 使用 cheerio 进行 DOM 内容解析
* wget 工具下载
* 文件/目录的保存和删除
